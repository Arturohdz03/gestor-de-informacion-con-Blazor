Comandos utilizados de DotNet

Crea la migraciones pendientes
dotnet ef migrations add NombreDeMiPrimeraMigracion

Actualizar la base de datos
dotnet ef database update

Elimina la base de datos
dotnet ef database drop

Listar todas las migraciones aplicadas
dotnet ef migrations list

Eliminar la última migración
dotnet ef migrations remove

Revertir la última migración
dotnet ef database update NombreDeLaMigracionAnterior
